== Description ==

**ACS Advanced Search** will add a advanced search link below the search field and support for advanced custom search and filter. You can expand your search to search within different taxonomies or categories or within custom post types, author, post date, post date range,post per page also add custom css in option field for advanced search form. This plugin is light, fast and coded with best practice.

**Features:**

* You can choose whether you want this plugin to add advanced search link at your search field or not.
* You can select number of posts show on per page.
* You can pick which post types you want your users to be able to select when performing a search otherwise search perform in all post types.
* You can pick which taxonomies you want your users to be able to select when performing a search.
* You can disable search input if you want.
* You can choose whether you want your users to be able to performing a search by post date or post date range.
* You can choose whether you want your users to be able to performing a search by author or not. 
* You can add your custom styling for advanced search form in option field.
* Included Shortcode



== Installation ==

They are two ways we can go about this: **Using the WordPress Plugin Search** or **Using the WordPress Admin Plugin Upload**.

Using the WordPress Plugin Search:

1. Go to WordPress admin area.
2. Click on Plugins > Add New.
3. In the Search box, type in **ACS Advanced Search** and hit enter.
4. Next to the **ACS Advanced Search** click on the Install Now button.
5. Click on Activate Plugin.

Using the WordPress Admin Plugin Upload:

1. Download the plugin here from our plugin page.
2. Go to WordPress admin area.
3. Click on Plugins > Add New > Upload Plugin.
4. Click on Choose File and select **acs-advanced-custom-search.zip**
5. Click on Install Now button.
6. Click on Activate Plugin.
